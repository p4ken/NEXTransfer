# p4ken/NEXTransfer

NEX-6 Wireless Transfer for Mac.

This project is not affiliated with Sony.

## Getting Started

1. Download this repository on your Mac.

1. Run `make` to launch the application.

1. Turn your NEX-6 into `View on Smartphone`[^1] mode.

1. Connect to NEX-6 from Mac via Wi-Fi.

1. Photos will be listed on the app.

[^1]: スマートフォン転送
